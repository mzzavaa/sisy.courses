# sisy.courses

Course catalog landing page for the SISY platform.

**Live site:** https://sisy.courses

## About

Online courses for women experts building visibility, mastering speaking craft, writing winning CFPs, and growing a powerful expert brand. Part of the [SISY ecosystem](https://sisy.world).

## Sections

- Hero - "Level up your expert voice"
- Course categories grid (Beginner / Intermediate / Advanced / Corporate)
- Featured course cards with pricing
- Pricing table (€49 - €499)
- Instructor spotlight
- Free Speaker Toolkit freebie section (marble gradient background)
- CTA

## Tech

Built with [Hugo](https://gohugo.io/). Deployed to GitHub Pages via Actions.

## Local development

```bash
hugo server -D
```

## Deployment

Auto-deploys on push to `main` via `.github/workflows/deploy.yml`.
